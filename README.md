# hikvision_device_scanner
Scan active HikVision devices in LAN. Retrieving Model, SN and Firmware.

1. Edit `hikvision.py` and change username and password for device to log in
2. Provide IP addresses that you wish to scan in `ips.txt` file
3. Run `hikvision.py` with `python3 hikvision.py`
4. Retrieved data will show, and saved to hikvision_devices.xlsx
